# Titanic Survival Prediction

## Project Overview
Machine Learning classification project that predicts whether a passenger survived the Titanic disaster.

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn

## ML Concepts
- Logistic Regression
- Train-Test Split
- Feature Engineering
- Accuracy Metrics

## Accuracy
Achieved 79% accuracy using Logistic Regression.

## Dataset
Titanic Dataset from Kaggle.
